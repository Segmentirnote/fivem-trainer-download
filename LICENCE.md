## 🧩 About

**FiveM Trainer** is a collection of open-source FiveM tools:

- **Mod Menu** — ESP, aimbot, vehicle spawner, money drop, god mode
- **Lua Executor** — run custom Lua scripts
- **Spoofer** — bypass HWID bans
- **Anti-Cheat Framework** — for server admins

## ⚙️ Configuration

Settings are stored in `config.json`.

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `menu_hotkey` | String | `"F2"` | Key to toggle menu |
| `process_name` | String | `"fivem.exe"` | Target process |
| `esp_enabled` | Boolean | `true` | Enable ESP |
| `aimbot_enabled` | Boolean | `false` | Enable aimbot |
